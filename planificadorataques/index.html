Instructivo de uso
1. Configura un borrador con raza, origen, destino, plaza de torneos y hora de llegada.
2. Edita las tropas en el panel derecho y pulsa Anadir para crear una fila en la matriz.
3. El borrador no se borra al anadir: puedes cambiar coordenadas, hora o tropas y volver a crear otra fila.
4. El planificador usa el map.sql del proyecto por defecto, pero tambien acepta uno pegado o cargado manualmente.
5. El recordatorio dispara sonido y notificacion cuando falten los segundos configurados para la hora de envio.
6. El link generado abre Send troops con coordenadas, tropas y eventType=3 para ataque normal.
